About the Table

The table was made from the OpenSky publications since 1/1/2014 having a DOI. Each of the 4600+ rows in the table represents a publication. The "pid" link opens a new window opened to the OpenSky landing page for that publication. The "doi" link opens a new window opened to the resolvedl doi location.

For each DOI, both WOS and Crossref web services were queried for grant information, namely award_ids. Note, sometimes the values for award_ids are not award_ids at all. If you click on an award_id, a new window is opened to a Kuali API client loaded with the award_id so you can see the API's response for that award_id.

Each award_id was submitted to the UCAR Kuali API. When grant information was found in Kuali, the award_id was added in the kuali_verified_award_ids column.
